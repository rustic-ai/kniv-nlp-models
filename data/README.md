# Training Data

## Universal Dependencies English EWT v2.14

- **Source:** https://universaldependencies.org/
- **License:** CC BY-SA 4.0
- **Tasks:** POS tagging, dependency parsing
- **Size:** ~16K sentences, ~254K tokens
- **Download:** `./download_ud.sh`

## CoNLL-2003 NER

- **Source:** https://www.clips.uantwerpen.be/conll2003/ner/
- **License:** Research use (Reuters corpus)
- **Tasks:** Named Entity Recognition (PER, ORG, LOC, MISC)
- **Size:** ~20K sentences
- **Download:** Automatic via HuggingFace Datasets
