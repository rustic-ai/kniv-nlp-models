# Conversation Domain

Agentic, task-oriented, and goal-driven conversations.

## Sources

| Source | License | Type | Expected Volume |
|--------|---------|------|-----------------|
| Taskmaster 1/2/3 | CC BY-4.0 | Task completion (booking, ordering, scheduling) | ~30K utterances |
| OASST1 | Apache 2.0 | Human-assistant multi-turn conversations | ~30K utterances |
| MultiWOZ 2.2 | Apache 2.0 | Multi-domain goal-driven (hotel, restaurant, train) | ~20K utterances |
| ABCD | MIT | Customer service with action-taking | ~15K utterances |
| AirDialogue | Apache 2.0 | Goal-driven flight booking | ~15K utterances |
| Glaive Function Calling | Apache 2.0 | Tool-use / function-calling conversations | ~15K utterances |
| Agent-FLAN | Apache 2.0 | Agent interaction traces | ~10K utterances |

## Characteristics

- Goal-driven dialog (scheduling, booking, ordering, problem-solving)
- Agent action-taking ("I've booked the room", "Let me check that")
- Tool use and function calling patterns
- Short to medium sentences (5-30 words)
- High pronoun frequency, informal register
- Named entities: people, locations, dates, times, monetary amounts
- Questions, requests, confirmations, plan commitments
- Multi-turn context dependencies
