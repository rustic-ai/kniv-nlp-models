# Conversation Domain

Conversational dialogue text — matches uniko's primary use case.

## Sources

| Source | License | Type | Expected Volume |
|--------|---------|------|-----------------|
| DailyDialog | CC BY-NC-SA 4.0 | Multi-turn dialogues | ~13K dialogues, ~100K utterances |
| OpenSubtitles (sample) | Free for research | Movie/TV subtitles | Sample of ~20K utterances |

## Characteristics

- Short sentences (avg 8-15 words)
- High pronoun frequency (he, she, they, it)
- Greetings, questions, fillers, reactions
- Informal language, contractions
- Named entities: people, places, dates
- Temporal expressions: "yesterday", "last week"
