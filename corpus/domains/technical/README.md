## Characteristics

- Technical terminology (API, database, algorithm, deployment, etc.)
- Procedural/instructional ("To configure X, set Y to Z")
- Code-adjacent prose (not code itself)
- Formal, precise language
- Named entities: software names, standards, protocols, organizations
