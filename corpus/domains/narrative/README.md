# Narrative Domain

Fiction and personal narrative text from public domain literature.

## Sources

| Source | License | Type | Expected Volume |
|--------|---------|------|-----------------|
| Project Gutenberg | Public domain | Classic fiction, novels | ~20K sentences from selected works |

## Characteristics

- Past tense narrative ("She went to the market")
- Named entities: character names, places, dates
- Temporal expressions: "the next morning", "last summer"
- Dialogue embedded in narrative
- Complex sentence structures, relative clauses
- Pronoun-heavy (coreference chains)
